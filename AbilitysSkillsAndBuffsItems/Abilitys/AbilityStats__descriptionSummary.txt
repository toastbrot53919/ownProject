The AbilityStats.cs file contains a class for managing the stats of an ability. It has the following fields: baseDamage, strengthScaling, intelligenceScaling, cooldown, manaCost (not implemented), range, projectileSpeed, and stunDuration. The class has a default constructor and three methods. setZero() sets all the fields to 0, addStats() adds the values of another AbilityStats object to the fields of this object, and printStats() returns a string that displays the non-zero fields of this AbilityStats object.
